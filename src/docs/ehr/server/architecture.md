# EHR server achitecture based on FHIR

- FHIR Server: Hapi Fhir
- DATABASE: PostgreSQL
- DICOM Server: Othanc
- Packages: fhirpy, sparc-me, Digitaltwins-api
- Environment: Docker, Python 3.9+
